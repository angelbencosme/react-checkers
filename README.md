## Introduction

This repository is a game of checkers made in react for the purpose of collaborating with the community.


## Features

- React 16.3.2.
- Create React App 3.
- React Scripts 1.1.4.
- React-Dom 16.3.2.

### Pre-Requisites

- Yarn 1.22.4
- Node 14.6.0
- Npm  6.14.6

### Installation

```bash
# If you use npm:
npm install

# Or if you use Yarn:
yarn install
```

**Run Checkers**

```bash
# If you use npm:
npm run start

# Or if you use Yarn:
yarn start
```
